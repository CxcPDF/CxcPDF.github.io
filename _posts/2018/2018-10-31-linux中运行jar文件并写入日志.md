1. 创建新文件

   ```shell
   touch log.log
   ```

2. 运行

   ``` shell
   nohup java -jar xxx.jar >log.log 2>&1 &
   ```

3. 查看进程

   ```shell
   ps -ef|grep xxx.jar
   ```

   <!--more-->

4. kill进程

   ```shell
   sudo kill -9 pid
   ```

5. 清空日志

   ```shell
   echo "">log.log
   ```
